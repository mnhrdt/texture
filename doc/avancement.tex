\section{Remarque}
\subsection{Création de mesh à partir de lidar}
Le lidar est bruité, le bord des immeubles n'est pas lisse. Le mesh naïf obtenu en reliant les points du lidar n'est donc pas satisfaisant.
L'utilisation de ``The Image Curvature Microscope'' sur ipol permet d'obtenir un lidar plus lisse à partir duquel on obtient un mesh plus propre. Plutôt que de créer les triangles toujours dans le même ordre il faudrait aussi que cela dépende de la hauteur des quatre points utilisés. Cela donnerait des surfaces plus lisses pour les immeubles en diagonal sur le lidar.
